Function


ฟังก์ชันในไพทอนแบ่งออกได้เป็น 2 ประเภท คือ

User-define function

1.User-define function คือ บล็อคการทำงานของโปรแกรมที่ถูกสร้างขึ้น เราเรียกบล็อคการทำงานนี้ว่า ฟังก์ชัน ซึ่งฟังก์ชันที่สร้างขึ้นจะยังไม่มีการทำงานใด ๆ จนกว่าจะมีส่วนของโค้ดโปรแกรมระบุเรียกฟังก์ชันเท่านั้น ฟังก์ชันจึงจะถูกเรียกให้ทำงาน
เราสามารถส่งค่าข้อมูลเข้าไปในฟังก์ชันได้ โดยฟังก์ชันจะกำหนดตัวแปรที่เรียกว่า "พารามิเตอร์" ขึ้นมา เพื่อใช้สำหรับรับข้อมูลเข้าไปในฟังก์ชัน
เมื่อฟังก์ชันทำงานเสร็จแล้ว จะส่งค่าข้อมูลกลับออกไปจากฟังก์ชันหรือไม่ก็ได้ ถ้าต้องการส่งข้อมูลกลับออกไปจากฟังก์ชัน ทำได้โดยใช้คำสั่ง return

 การสร้างฟังก์ชัน

สามารถสร้างฟังก์ชันได้ด้วยรูปแบบดังต่อไปนี้
 def ชื่อฟังก์ชัน() :   คำสั่งการทำงานต่าง ๆ ภายในฟังก์ชัน

 การเรียกฟังก์ชันให้ทำงาน

สามารถเรียกฟังก์ชันให้ทำงานได้ด้วยรูปแบบดังต่อไปนี้
 ชื่อฟังก์ชัน ()

ตัวอย่าง การสร้างฟังก์ชันและเรียกใช้งานฟังก์ชัน

Code ผลลัพธ์
def greeting() :
  print("สวัสดีค่ะ ยินดีต้อนรับ")
  print("ขอให้มีความสุขกับบริการของเรา")
 greeting(): 		  สวัสดีค่ะ ยินดีต้อนรับ
  ขอให้มีความสุขกับบริการของเรา

 การกำหนดพารามิเตอร์ของฟังก์ชัน

พารามิเตอร์
คือ ตัวแปรที่ใช้รับข้อมูลเข้ามาในฟังก์ชัน การกำหนดพารามิเตอร์ของฟังก์ชันทำได้ดังรูปแบบดังต่อไปนี้
 def ชื่อฟังก์ชัน (พารามิเตอร์ตัวที่1, พารามิเตอร์ตัวที่2,...,พารามิเตอร์ตัวสุดท้าย):
  คำสั่งการทำงานต่าง ๆ ภายในฟังก์ชัน

ตัวอย่าง โปรแกรมรับพารามิเตอร์เข้ามาในฟังก์ชันเพื่อหาผลบวก

Code ผลลัพธ์
def plus(num1, num2):
  print("num 1 = ",num1)
  print("num 2 = ",num2)
  print("num1+num 2 = ",num1+num2)
plus(1,2)
plus(3,4) 		num 1 = 1
num 2 = 2
num1+num2 = 3
num 1 = 3
num 2 = 4
num1+num2 = 7

 การกำหนดค่า default ของพารามิเตอร์ในฟังก์ชัน

ค่า default
คือ ค่าที่เรากำหนดไว้แล้ว โดยเราสามารถกำหนดค่า default ให้กับพารามิเตอร์ของฟังก์ชันได้ ในกรณีที่ไม่มีการส่งค่าข้อมูลใดเข้ามาในฟังก์ชัน พารามิเตอร์นั้นจะถูกกำหนดค่าเป็นค่า default
การกำหนดค่า default ของพารามิเตอร์ ทำได้ดังรูปแบบดังต่อไปนี้
 def ชื่อฟังก์ชัน (พารามิเตอร์ตัวที่ 1 = ค่า default,...,พารามิเตอร์ตัวสุดท้าย = ค่า default):
  คำสั่งการทำงานต่าง ๆ ภายในฟังก์ชัน

ตัวอย่าง โปรแกรมแสดงการกำหนดค่า default ของพารามิเตอร์
Code ผลลัพธ์
def information(nationality="ไทย"):
  print("ฉันสัญชาติ"+nationality)
information("ไทย")
information("อินเดียน")
information("ออสเตรเลียน")

Built-in functions

 2.Built-in functions คือ ฟังก์ชันที่ Python interpreter ได้จัดเตรียมไว้ให้เรียกใช้งาน ซึ่งมีอยู่มากมายหลายฟังก์ชัน ตัวอย่างฟังก์ชันที่เราได้ศึกษาไปก่อนหน้านี้แล้วได้แก่ input() print() int() float() complex() str() tuple() set() dict() และ range() เป็นต้น
ในทีนี้จะขอกล่าวถึงฟังก์ชันอื่น ๆ ที่ถูกใช้งานบ่อย ดังนี้


ชื่อฟังก์ชัน ความหมาย ตัวอย่าง ผลลัพธ์
abs() คืนค่าสัมบูรณ์ของตัวเลข abs(-3) 3
max() คืนค่ามากสุดของตัวเลข max(20,10) 20
min() คืนค่าน้อยสุดของตัวเลข min(20,10) 10
pow() คืนค่าเลขยกกำลัง pow(3,2) 9
round() ปัดเศษตัวเลขตามจำนวนหลักที่กำหนด round(25.43985,2) 25.44
sum() คืนค่าผลรวม x = (1,2,3)
print(sum(x))		 6
bin() คืนค่าเลขฐานสอง bin(10) 0b1010
hex() คืนค่าเลขฐานสิบหก hex(10) 0xa
oct() คืนค่าเลขฐานแปด oct(10) 0o12
bool() คืนค่าบูลีน bool(1<2)< /td> True
type() คืนค่าว่าเป็นชนิดข้อมูลประเภทใด type(123) (class'int')
len() คืนค่าความยาวของข้อมูลหรือจำนวนข้อมูลว่ามีกี่ตัว len("Python") 6

โมดูล (Module)


 โมดูล คือ ไฟล์โปรแกรมที่ถูกสร้างขึ้นเขึ้นมาแยกต่างหากจากไฟล์โปรแกรมหลัก และจะถูกเรียกใช้งานจากโปรแกรมหลักอีกทีหนึ่ง ซึ่งไฟล์นี้จะมีนามสกุล .py เหมือนกับโปรแกรมหลัก โมดูลหนึ่ง ๆ จะประกอบด้วยฟังก์ชัน คลาส หรือตัวแปรกี่ตัวก็ได้ โดยปกติหากมีงานใด ๆ ก็ตามที่มักจะถูกเรียกทำงานซ้ำ ๆ หรือใช้งานบ่อย ๆ เราจะแตกงานนั้นออกมาเป็นฟังก์ชัน และนำฟังก์ชันต่าง ๆ ที่เกี่ยวข้องกันมารวมเข้าด้วยกันเป็นโมดูล ตัวอย่างของโมดูล เช่น โมดูลการวาดรูปกราฟฟิก โมดูลการคำนวณตัวเลข โมดูลคำนวณพื้นที่เรขาคณิตเป็นต้น

 วิธีการสร้างโมดูล

ให้สร้างฟังก์ชันการทำงานใด ๆ ตามที่ต้องการ จากนั้นบันทึกไฟล์ให้มีนามสกุล .pv ตัวอย่างเช่น ร้านอาหารแห่งหนึ่งเปิดให้บริการอาหารไทยและญี่ปุ่น เมนูอาหารจะเปลี่ยนเดือนละ 1 ครั้ง เมื่อลูกค้าต้องการสั่งอาหารจะต้องแสดงเมนูอาหารทั้ง 2 ประเภททุกครั้ง ดังนั้นเราจึงสร้างโมดูลเมนูอาหารข้นมาดังนี้

 def thaimenu()

 วิธีการเรียกใช้งานโมดูล

1. ใช้คำสั่ง Import
เป็นการโหลดโมดูลเข้ามายังโปรแกรมหลักก่อนเรียกใช้งานฟังก์ชันต่าง ๆ ในโมดูลนั้น โดยมีรูปแบบดังนี้
 import ชื่อโมดูล
  ชื่อโมดูล.ชื่อฟังก์ชันในโมดูล

2. ใช้คำสั่ง from ... import
เป็นการเลือกโหลดเฉพาะฟังก์ชันที่ต้องการใช้งานของโมดูลเข้ามายังโปรแกรมหลัก ก่อนเรียกใช้ฟังก์ชันนั้น ๆ ให้ทำงาน โดยมีรูปแบบดังนี้
 from ชื่อโมดูล import ชื่อฟังก์ชันที่ต้องการใช้งาน
