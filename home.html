Learning Python

ทำไมควรเลือกเขียนโปรแกรมด้วยภาษาไพทอน

  • ทำงานได้หลายระบบปฏิบัติการ

    • โปรแกรมภาษาไพทอนสามารถทำงานบนระบบปฏิบัติการต่าง ๆ ได้ ไม่ใช่ว่าเขียนโปรแกรมไพทอนบน Windows แล้วจะต้องใช้งานบน Windows เท่านั้น

  • Open-source

    • ไพทอนเป็น Open source ที่ใครก็ได้สามารถดาวน์โหลดไปใช้งานได้ฟรี หรือแก้ไขเพิ่มเติมได้โดยไม่มีค่าใช้จ่าย

  • เข้าใจง่าย

    • โครงสร้างของโปรแกรมภาษาไพทอนไม่ซับซ้อน คำสั่งเข้าใจง่าย เหมาะกับผู้ที่ต้องการเริ่มเขียนโปรแกรม

  • Object Oriented

    • ไพทอนสนับสนุนการพัฒนาโปรแกรมเชิงวัตถุ (Object Oriented Programming)

  • Library

    • ไพทอนมีไลบรารีจำนวนมากให้เรียกใช้งานได้ เช่น ไลบรารีเกี่ยวกับการทำงานด้านคณิตศาสตร์ ด้านกราฟฟิก ปัญญาประดิษฐ์ เป็นต้น

  • ใช้ได้หลาย Database

    • ไพทอนมีฟังก์ชันที่สนับสนุนการทำงานกับฐานข้อมูลประเภทต่าง ๆ เช่น MySQL Oracle Informix เป็นต้น

ทำความรู้จักกับภาษา Python

01

ภาษาคอมพิวเตอร์คือ ?

มนุษย์ทุกคนบนโลกสื่อสารเข้าใจกันด้วย "ภาษามนุษย์" แต่คอมพิวเตอร์ไม่เข้าใจภาษามนุษย์ การที่มนุษย์จะสื่อสาร กับคอมพิวเตอร์ให้เข้าใจ จะต้องใช้ ภาษาคอมพิวเตอร์ เท่านั้น โดยไพทอนก็นับว่าเป็นภาษาคอมพิวเตอร์ประเภทหนึ่ง

02

ตัวแปลภาษาคือ ?

ภาษามนุษย์แบ่งออกเป็นหลากหลายภาษา เช่น ภาษาไทย ภาษาจีน ภาษาฝรั่งเศส ภาษาญี่ปุ่น เป็นต้น ในบางครั้งหากไม่เข้าใจในภาษานั้น ๆ ก็ต้องอาศัยล่ามเป็นผู้ช่วยแปลภาษา เพื่อแปลผลให้เข้าใจกัน

สรุป

ภาษาคอมพิวเตอร์ก็เช่นเดียวกับภาษามนุษย์ คือมีหลายภาษาที่สามารถใช้สื่อสารกับคอมพิวเตอร์ได้ บางภาษาสื่อสารแล้วคอมพิวเตอร์ก็เข้าใจได้เลย แต่บางภาษาก็ไม่ใช่ภาษาที่คอมพิวเตอร์เข้าใจ

ดังนั้น ภาษาคอมพิวเตอร์บางตัวจึงต้องใช้ตัวแปลภาษา เป็นล่ามในการทำหน้าที่แปลจากภาษาคอมพิวเตอร์ไปเป็นภาษาที่เครื่องคอมพิวเตอร์เข้าใจ ว่าต้องการให้คอมพิวเตอร์ทำงานอะไร
