 ตัวแปร (Variables) คือ ชื่อที่กำหนดขึ้นเพื่อใช้อ้างอิงค่าข้อมูลบนหน่วยความจำ ซึ่งค่าข้อมูลนี้สามารถนำไปเรียกใช้ในโปรแกรมได้ ทั้งนี้การประกาศชื่อตัวแปรขึ้นใช้งานจะเป็นไปตามกฏการตั้งชื่อของตัวแปร

การกำหนดค่าให้กับตัวแปร


 การกำหนดค่าให้กับตัวแปรในภาษาไพทอนมีรูปแบบดังนี้

ชื่อตัวแปร = ค่าที่ต้องการกำหนดให้ตัวแปล

ตัวอย่างเช่น
  price = 100
คำสั่งนี้จะมีผลทำให้ตัวแปร price ถูกประกาศขึ้น และถูกกำหนดให้มีชนิดข้อมูลเป็นเลขจำนวนเต็ม (Integer) หรือเรียกสั้น ๆ ว่า int โดยอัตโนมัติ ซึ่งตัวแปร price จะชี้อ้างอิงไปยังตำแหน่งบนหน่วยความจำที่เก็บค่าเลขจำนวนเต็ม 100 อยู่
  name = "Orapin"
คำสั่งนี้จะมีผลทำให้ตัวแปร name ถูกประกาศขึ้น และถูกกำหนดให้เป็น string (สตริง) โดยอัตโนมัติ ซึ่งตัวแปร name จะชี้อ้างอิงไปยังตำแหน่งบนหน่วยความจำที่เก็บค่าข้อมูล Orapin อยู่


การกำหนดค่าให้กับตัวแปรหลายตัว


ในภาษาไพทอนเราสามารถค่า 1 ค่าให้กับตัวแปรหลาย ๆ ตัวพร้อมกันได้ เช่น
  a = b = c = 100
หมายถึง กำหนดให้ตัวแปร a b และ c ชี้อ้างอิงไปยังตำแหน่งบนหน่วยความจำที่เก็บค่าข้อมูลเลขจำนวนเต็ม 10 อยู่
  name = "Orapin"
คำสั่งนี้จะมีผลทำให้ตัวแปร name ถูกประกาศขึ้น และถูกกำหนดให้เป็น string (สตริง) โดยอัตโนมัติ ซึ่งตัวแปร name จะชี้อ้างอิงไปยังตำแหน่งบนหน่วยความจำที่เก็บค่าข้อมูล Orapin อยู่

ใช้ฟังก์ชัน input() รับข้อมูลเก็บลงในตัวแปร


 เมื่อใช้ฟังก์ชัน input() รับค่าข้อมูลจากผู้ใช้งานโปรแกรมแล้ว เราสามารถนำค่านั้นมาจัดเก็บลงในตัวแปรได้ ดังนี้

ชื่อตัวแปร = input(ข้อความที่จะแสดงก่อนรับค่าข้อมูล)


การเปลี่ยนแปลงค่าของตัวแปร


 เมื่อมีการกำหนดค่าให้กับตัวแปร ตัวแปรนั้นก็จะถูกสร้างขึ้นพร้อมทั้งถูกกำหนดชนิดข้อมูลให้โดยอัตโนมัติตามข้อมูลที่ตัวแปรนั้นชี้อ้างอิงอยู่
แต่หากต่อมามีการกำหนดค่าใหม่ให้กับตัวแปร ชนิดข้อมูลของตัวแปรนั้นก็จะเปลี่ยนแปลงไปตามค่าใหม่ที่กำหนดให้

คำสั่ง ผลลัพธ์
x = 100
print(x)		 100
y = x
print(y)		 100
y = 200
print(y)		 200
y = "abc"
print(y)		 abc
