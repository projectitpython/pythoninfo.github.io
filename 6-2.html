คำสั่งเงื่อนไข if


 คำสั่ง if ใช้ตรวจสอบเงื่อนไขหนึ่ง ๆ หากเงื่อนไขเป็นจริงแล้วคำสั่งภายในบล็อคของเงื่อนไข if จะถูกประมวลผล ในที่นี้คือคำสั่งที่ 1 แต่ถ้าเงื่อนไขเป็นเท็จแล้วคำสั่งที่อยู่ภายในบล็อคของเงื่อนไข if จะไม่ได้รับการประมวลผล คือจะข้ามไปประมวลผลคำสั่งที่อยู่นอกบล็อคของ if ทันที ในที่นี้คือคำสั่งที่ 2

ตัวอย่าง โปรแกรมตรวจสอบอายุโดยใช้คำสั่ง if


Code ผลลัพธ์
age = input("กรุณากรอกอายุ : ")
  if int(age)<20:
    print("คุณยังเป็นผู้เยาว์")
    print("คุณอายุ "+age+"ปี")
   กรุณากรอกอายุ : 20
  คุณอายุ 20 ปี

  กรุณากรอกอายุ : 18
  คุณยังเป็นผู้เยาว์
  คุณอายุ 18 ปี

คำสั่งเงื่อนไข if-else


 เป็นคำสั่งที่ช่วยให้การตรวจสอบเงื่อนไขสมบูรณ์ขึ้น กล่าวคือ ถ้าเงื่อนไขของคำสั่ง if เป็นจริงแล้วคำสั่งภายในบล็อคของเงื่อนไข if จะถูกประมวลผลในทีนี้คือคำสั่งที่ 1 แต่ถ้าเงื่อนไขของคำสั่ง if เป็นเท็จแล้วคำสั่งภายในบล็อคของ else ก็จะถูกประมวลผลแทน ในทีนี้คือคำสั่งที่ 2 เมื่อจบการทำงานของคำสั่งเงื่อนไข if-else ก็จะทำงานตามคำสั่งที่อยู่นอกบล็อก if-else ต่อไป ในทีนี้คือคำสั่งที่ 3

ตัวอย่าง โปรแกรมตรวจสอบอายุโดยใช้คำสั่ง if-else

Code ผลลัพธ์
age = input("กรุณากรอกอายุ : ")
  if int(age)<20:
    print("คุณยังเป็นผู้เยาว์")
  else :
    print("คุณบรรลุนิติภาวะแล้ว")
print("คุณอายุ "+age+"ปี")		   กรุณากรอกอายุ : 20
  คุณบรรลุนิติภาวะแล้ว
  คุณอายุ 20 ปี

  กรุณากรอกอายุ : 18
  คุณยังเป็นผู้เยาว์
  คุณอายุ 18 ปี

คำสั่งเงื่อนไข if-elif-else


 การใช้คำสั่ง if-elif-else ช่วยให้การตรวจสอบเงื่อนไขสมบูรณ์มากยิ่งขึ้น เนื่องจากสามารถตรวจสอบได้หลายเงื่อนไข หากตรวจสอบเงื่อนไขที่ 1 แล้วเป็นจริง ก็จะทำงานตามคำสั่งที่ 1 แต่ถ้าเงื่อนไขเป็นเท็จ จะตรวจสอบเงื่อนไขที่ 2 ต่อ ถ้าเงื่อนไขที่ 2 เป็นจริง จะทำคำสั่งที่ 2 แต่ถ้าเป็นเท็จ จะไปตรวจสอบเงื่อนไขที่ 3 ถ้าเงื่อนไขที่ 3 เป็นจริง จะทำคำสั่งที่ 3 แต่ถ้าเป็นเท็จ จะทำคำสั่งที่ 4 ทันทีเพราะไม่มีเงื่อนไขใด ๆ ให้ตรวจสอบแล้ว (จะพบว่าถ้าไม่มีเงื่อนไขใดเป็นจริงเลย ก็จะทำงานในคำสั่งที่ 4 ทุกครั้ง) และเมื่อตรวจสอบเงื่อนไขและประมวลผลตามคำสั่งของ if-elif-else เรียบร้อยแล้ว ก็จะทำงานในคำสั่งที่ 5 ต่อไป

ตัวอย่าง โปรแกรมประมวลผลเกรดโดยใช้คำสั่ง if-elif-else
Code ผลลัพธ์
score = input("กรุณากรอกคะแนน : ")
  if int(score)>=80 :
    print("คุณได้เกรด A")
  elif int(score) >=70 :
    print("คุณได้เกรด B")
  elif int(score) >=60 :
    print("คุณได้เกรด C")
  elif int(score) >=50 :
    print("คุณได้เกรด D")
  else :
    print("คุณได้เกรด F")		   กรุณากรอกคะแนน : 89
  คุณได้เกรด A

  กรุณากรอกคะแนน : 49
  คุณได้เกรด F
