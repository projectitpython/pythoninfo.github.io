บทที่ 9 Exception

  1. ย้อนกลับ
  2. หน้าถัดไป

Exception คือ


 Exception คือ เหตุการณ์ผิดปกติที่เกิดขึ้นระหว่างการประมวลผลของโปรแกรม ส่งผลให้โปรแกรมทำงานไม่เป็นไปตามขั้นตอนที่ได้กำหนดไว้
ตัวอย่างของ Exception เช่น
 - โค้ดของโปรแกรมมีการหารตัวเลขหนึ่ง ๆ ด้วยศูนย์ (divide by zero)
 - โปรแกรมพยายามเปิดไฟล์ที่ไม่มีอยู่จริง
 - โปรแกรมต้องการรับอินพุตเป็นเลขจำนวนเต็ม แต่ผู้ใช้โปรแกรมป้อนข้อมูลเข้ามาเป็นสตริง

วิธีจัดการกับ Exception


จัดการกับ Exception ด้วย try-except

 การดักจับ exception สามารถทำได้โดนง่าย เพียงแค่กำหนดบล็อคของ try-except ครอบส่วนของโปรแกรมที่คิดว่าอาจก่อให้เกิด exception ขึ้นเท่านั้น
รูปแบบของ try-except คือ
 try:
  ส่วนของโปรแกรมที่อาจก่อให้เกิด Exception ขึ้น
 except exception1 :
  สิ่งที่จะทำให้โปรแกรมทำเมื่อดักจับ exception ที่ 1 ได้
 except exception2 :
  สิ่งที่จะทำให้โปรแกรมทำเมื่อดักจับ exception ที่ 2 ได้
  ...
 except exception n :
  สิ่งที่จะทำให้โปรแกรมทำเมื่อดักจับ exception ตัวสุดท้ายได้

จัดการกับ Exception ด้วย try-except-else

 เราสามารถนำคีย์เวิร์ด else มาใช้ร่วมกับการดักจับ Exception ได้ โดยถ้าตรวจสอบไม่พบ Exception ใด ๆ โปรแกรมจะทำงานตามคำสั่งที่ระบุไว้ในส่วนของ else
รูปแบบของ try-except-else คือ
 try:
  ส่วนของโปรแกรมที่อาจก่อให้เกิด Exception ขึ้น
 except exception1 :
  สิ่งที่จะทำให้โปรแกรมทำเมื่อดักจับ exception ที่ 1 ได้
 except exception2 :
  สิ่งที่จะทำให้โปรแกรมทำเมื่อดักจับ exception ที่ 2 ได้
  ...
 except exception n :
  สิ่งที่จะทำให้โปรแกรมทำเมื่อดักจับ exception ตัวสุดท้ายได้
 else:
  สิ่งที่จะให้ทำเมื่อตรวจสอบไม่พบ exception

จัดการกับ Exception ด้วย try-except-finally

 เราสามารถนำคีย์เวิร์ด finally มาใช้ร่วมกับการดักจับ exception ไดเ โดยโปรแกรมจะทำงานตามคำสั่งที่ระบุไว้ในส่วนของ Finally เสมอไม่ว่าจะตรวจสอบพบ exception หรือไม่ก็ตาม
รูปแบบของ try-except-finally คือ
 try:
  ส่วนของโปรแกรมที่อาจก่อให้เกิด Exception ขึ้น
 except exception1 :
  สิ่งที่จะทำให้โปรแกรมทำเมื่อดักจับ exception ที่ 1 ได้
 except exception2 :
  สิ่งที่จะทำให้โปรแกรมทำเมื่อดักจับ exception ที่ 2 ได้
  ...
 except exception n :
  สิ่งที่จะทำให้โปรแกรมทำเมื่อดักจับ exception ตัวสุดท้ายได้
 finally:
  สิ่งที่จะให้ทำเมื่อตรวจสอบพบหรือไม่พบ exception

จัดการกับ Exception ด้วย try-except-raise

 นอกจากการจัดการกับ exception ด้วย try-except แล้ว เราสามารถตรวจสอบเงื่อนไขใด ๆ ในโปรแกรมด้วยตัวเอง เมื่อตรวจสอบพบข้อผิดพลาดก็สามารถใช้คีย์เวิร์ด raise แสดงข้อผิดพลาดแจ้งเตือนได้ด้วยตัวเอง
รูปแบบของ try-except-raise คือ
 raise Exception(msg[args,[,traceback]])

  • Exception คือ ประเภทของความผิดพลาด
  • msg คือ ข้อความอธิบายข้อผิดพลาด
  • args คือ ค่าอาร์กิวเมนต์ที่ส่งไปยังคลาส Exception
  • traceback คือ สาเหตุที่ทำให้เกิดข้อผิดพลาด
